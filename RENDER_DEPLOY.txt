# Deploy Backend to Render.com
# Free Python service hosting with GitHub integration

## Prerequisites:
# 1. Create account at https://render.com
# 2. Install Render CLI: pip install render
# 3. Authenticate: render login

## Steps:
# 1. In Render dashboard, create new "Web Service"
# 2. Connect your GitHub repository
# 3. Configure:
#    - Name: kalmskills-backend
#    - Runtime: Python 3.11
#    - Build command: pip install -r backend/requirements.txt
#    - Start command: cd backend && gunicorn -w 4 -b 0.0.0.0:8000 main:app
#    - Select Free tier

## After deployment:
# - Backend URL will be: https://kalmskills-backend.render.com
# - Update .env.production with this URL
